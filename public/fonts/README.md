# SVN-Determination Sans Font Setup

## Cách tải và cài đặt font SVN-Determination Sans:

1. **Tải font từ nguồn chính thức:**
   - Truy cập: https://www.svnfont.com/viet-hoa-svn-determination-sans/
   - Tải file font về máy

2. **Chuyển đổi font sang định dạng web:**
   - Sử dụng công cụ như Font Squirrel Webfont Generator
   - Hoặc sử dụng online converter để chuyển từ TTF sang WOFF2/WOFF

3. **Thay thế các file placeholder:**
   - Thay thế `SVN-DeterminationSans-Regular.woff2` bằng file font thực
   - Thay thế `SVN-DeterminationSans-Bold.woff2` bằng file font thực
   - Thêm các file WOFF làm fallback nếu cần

4. **Cấu trúc file cần có:**
   ```
   public/fonts/
   ├── SVN-DeterminationSans-Regular.woff2
   ├── SVN-DeterminationSans-Regular.woff
   ├── SVN-DeterminationSans-Bold.woff2
   ├── SVN-DeterminationSans-Bold.woff
   └── README.md
   ```
